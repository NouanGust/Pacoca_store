PaçocaStore API

Documentação oficial para consumo de dados de produtos.

Sobre a API

Esta API foi criada para fornecer dados de produtos para a construção de projetos de e-commerce. Ela retorna dados no formato JSON.

URL Base: https://pacoca-store.onrender.com

Endpoints Disponíveis

1. Listar todos os produtos

Retorna a lista completa de produtos cadastrados na loja.

  • Método: GET
  • Rota: /api/produtos

Exemplo de Resposta (200 OK):

[
  {
    "id": 1,
    "nome": "Smartphone Galaxy S23",
    "preco": 3500.0,
    "categoria": "eletronicos",
    "descricao": "Smartphone de última geração...",
    "imagem": "https://placehold.co/400"
  }
]

2. Filtrar produtos por categoria

Você pode filtrar os produtos passando o parâmetro categoria na URL (Query Parameter).
Categorias disponíveis: eletronicos, eletrodomesticos, roupas, livros, diversos.

  • Método: GET
  • Rota: /api/produtos?categoria=nome_da_categoria
  • Exemplo de Requisição: /api/produtos?categoria=eletronicos

3. Buscar um produto específico pelo ID

Retorna os detalhes de um único produto com base no seu número de identificação (ID). Útil para criar a página de "Detalhes do Produto".

  • Método: GET
  • Rota: /api/produtos/{id}
  • Exemplo de Requisição: /api/produtos/3

Exemplo de Resposta (200 OK):

{
  "id": 3,
  "nome": "Camiseta Algodão Básica",
  "preco": 49.9,
  "categoria": "roupas",
  "descricao": "Camiseta 100% algodão, cor preta, tamanho M.",
  "imagem": "https://placehold.co/400"
}
